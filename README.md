# 广小轻小程序后端
是小轻呀
## Getting Started

前置要求：
本机安装了 jdk8 或更高版本，并确保下载好pom文件的所需依赖项

如何运行程序：
数据使用本地的:
使用doc/sql的devqing重新部署文档.sql既可以重新部署sql 

数据不使用本地的:
直接运行prod环境

​		Java ，SpringBoot ，SpringMVC ，MyBatis-Plus ，MySQL ，Redis，
